Welcome to Frontiers in Plant Sciences 2016
Tutorials in computational molecular evolution

Herewith you will find a set of exercises/step-by-step guides on common problems in molecular evolution as:

  1. Inferring phylogenetic trees
  2. Detecting natural selection on protein coding genes

You can download the entire repository containing exercises, datasets, step-by-step solutions and results from the Github repository
